# Item Management System

This is a simple Java CRUD project. I built it with Servlet, JSP, and MySQL.

## Features
- Add Item
- Update Item
- View Item Details
- Delete Item

## Setup
1. Run `database.sql` in MySQL to create the database and table.
2. Update the DB password in `ItemDAO.java` if needed. (Right now it's set to "1234")
3. Open in VS Code with Java extensions and Maven support.
4. Run using Tomcat extension.
5. Go to `http://localhost:8080/TEST/`
